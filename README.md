# 病院経営指標ダッシュボード

### 財務データから経営指標を計算し、グラフ化するwebアプリです

**ver 0.2 β版**　　2026.7.19
**ver 0.1 β版**　　2026.7.7

### **特徴**  
   **財務データ**から、収益性・効率性・安全性の高度なKPIを自動計算  
   **時系列分析モード**で、年度推移を折れ線・棒グラフにて描画可能に  
   **相関分析モード**で、「固定比率」×「ROA」などの戦略的なポジション分析が可能  
   **自由に指標を追加可能に**：データファイル(.xlsx)の別シートに定義すれば反映されます 

### **方法**  
１．【準備】以下の構成のエクセルファイルを作成してください(.xlsx形式)  
  #### **data**シート："年度"（西暦４桁）、"施設名"、"勘定科目"、"金額"の4項目  
| 年度 | 施設名 | 勘定科目 | 金額 |
| :--- | :--- | :--- | :--- |
| 2024 | 焼津市立 | 医業収益 | 3981200000 |
    ※列名項目の名称が上記と異なるとエラーになります  
    
  #### **subjects**シート：入力データを定義します。  
    "科目名"：データ名称  
    "種類"：データのカテゴリ(financial/functional)  （必須）  
    "番号"：参照先と表示順（PLカテゴリの表示順、など）（必須）  
    "備考"：科目名が属する項などを記載していま項（任意）  
    "計算"：指標の場合の計算式です（任意）  
  
  #### **kpi_categories**：指標（計算）を定義します。すべて必須項目です。  
    "カテゴリ"：指標をまとめる分類項目です（収益性/効率性/安全性/活動性/生産性）  
    "KPI名"：指標の名称  
    "計算式"：解説で表示される計算式です  
    "単位"：指標の単位です。グラフ軸でも使用されます。  
    "経営的な意味"：グラフの下に表示される解説です。  
    "判断の目安・基準"：グラフの下に表示される解説です。  
    "計算ロジック"：プログラム内で計算する式です。  
      A/B×100[%}の場合：(df['A'] / df['B'].replace(0, np.nan)) * 100　と入力  
    
  #### **units**：科目名/指標名に含まれる単語を指定することで、単位を取得します。(必須)  
    
  #### **ピボット(確認用)：こちらは任意のｼｰﾄです（なくてもよい）  
    

２．リンク先からweb画面を開き、画面左のアップロードエリアへファイルをアップしてください  
    
３．画面右側の「アップロードデータの要約」のブルダウンメニューを開き、データ欠損がないか確認できます  
  **赤**：データが0の場合  
  **黄**：一部データが欠損している年度があります  
  **緑**：すべての年度にデータ入力されています  
    
４．２つのタブを切り替えて使用できます  
　　・「単一指標の時系列比較」：指標を選択すると時系列の折れ線/棒グラフが作成されます  
      ●左メニューで折れ線/棒グラフの選択が可能です。  
　　・「2指標相関分析」: 指標または財務データをx軸, y軸に設定すると年度別のドット（散布図）が作成されます  
    　●左メニューで「単一指標の時系列比較」/「2指標相関分析」の切り替えが可能です  
    
※ データは一切保存されませんので、再度使用する場合は.xlsxデータのアップロードが必要です
