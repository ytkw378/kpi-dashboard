# 病院経営指標ダッシュボード

### 財務データから経営指標を計算し、グラフ化するwebアプリです

**ver 0.1 β版**　　2026.7.7

### **特徴**  
   **19項目の財務データ**から、収益性・効率性・安全性の高度なKPIを自動計算  
   **スマートな2段階選択UI**で、目的の指標へ迷わずアクセス  
   **相関分析モード**で、「固定比率」×「ROA」などの戦略的なポジション分析が可能  

### **財務データ(勘定項目)** 以下のデータを用います
   医業収益 / 給与費 / 材料費 / 経費 / 減価償却費 / 医業利益 / 経常利益  
   固定資産 / 貯蔵品 / 流動資産 / 当座資産 / 総資産 / 固定負債 / 医業未収金  
   流動負債 / 純資産 / 委託費 / 医薬品費 / 未収金 / 院外処方率

**方法**  
１．以下のテーブルを作成してください(.xlsx形式)  
　　　列名称：「年度」（西暦４桁のみ）、「施設名」、「勘定科目」、「金額」（円）の4項目  
  
２．画面左のアップロードエリアへファイルをアップしてください  
  
３．画面右側の「アップロードデータの要約」のブルダウンメニューを開き、データ欠損がないか確認できます  
  
４．２つのタブを切り替えて使用できます  
　　・「単一指標の時系列比較」：指標を選択すると時系列の折れ線グラフが作成されます  
　　・「2指標相関分析」: 指標または財務データをx軸, y軸に設定すると年度別のドット（散布図）が作成されます  
  
※ データは一切保存されませんので、再度使用する場合は.xlsxデータのアップロードが必要です
